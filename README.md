# Proyecto Eimy

Sitio web desarrollado.  
Incluye estructura HTML5 semántica, estilos CSS con Flexbox y control de versiones con Git y GitHub.

- `index.html`: Página principal con secciones de inicio, galería y formulario de contacto.
